# ESP32_Button_Box
This is my button box for SCS truck games

## How to use?
1. Download required libraries.
2. TFT_eSPI requires some modifications in your downloaded "User_Setup.h" file. Search where your library is (probably in "Arduino/TFT_eSPI/") and you replace your "User_Setup.h" by the one in "[config/User_Setup.h](https://github.com/AndersonRezende/ESP32_Button_Box/blob/master/config/User_Setup.h)".
